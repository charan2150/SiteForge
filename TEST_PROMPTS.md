# SiteForge Test Prompts

Use these 10 prompts to test the SiteForge tool. Draw a polygon on the map, paste the prompt, then "Enhance Prompt" → "Generate Visualization". Verify overlay alignment and output quality.

---

## 1. Construction excavation
```
Large open-cut excavation with steel sheet piling, exposed brown earth, two excavators, concrete formwork at base, construction barriers around perimeter
```

## 2. Completed development
```
New residential development with finished townhouses, paved parking lot, landscaping with trees and lawns, street lighting
```

## 3. Solar farm
```
Large-scale solar panel array in neat rows, inverter station, perimeter fencing, gravel access roads between rows
```

## 4. Park redesign
```
Community park with soccer field, playground, walking paths, picnic pavilion, mature trees along perimeter
```

## 5. Parking lot retrofit
```
Parking lot with EV charging stations, permeable paver surface, bioswales for stormwater, bike racks
```

## 6. Landfill / remediation
```
Capped landfill with geomembrane cover, methane collection wells, monitoring equipment, vegetation establishment
```

## 7. Bridge abutment
```
Bridge construction with drilled shafts, formwork for abutment, rebar cages, crane and equipment staging
```

## 8. Green roof
```
Rooftop garden with sedum mat, raised planters, irrigation system, walking paths, seating area
```

## 9. Quarry / mining
```
Active quarry with tiered benches, haul roads, crushing plant, sediment pond, safety berms
```

## 10. Wetland restoration
```
Constructed wetland with native vegetation, inlet/outlet structures, boardwalk for access, interpretive signage
```

---

**Testing checklist**
- [ ] No zoom glitch during capture
- [ ] Overlay aligns with base map (no shift or ghosting)
- [ ] Red polygon area edits correctly; outside area unchanged
- [ ] Output matches satellite style (straight-down, photorealistic)
